# Programming FAQ Chatbot 🤖

A modern, interactive React-based chatbot designed to answer theoretical programming questions. It features a sleek user interface, quick-reply suggestion chips, and an intelligent fuzzy-matching system to understand user intent without requiring exact phrasing.

## ✨ Features

- **Intelligent Keyword Matching:** Uses stop-word filtering and fuzzy matching to understand what the user is asking, even if they don't type the exact dictionary key.
- **Quick Reply Suggestions:** Pre-ask buttons integrated into the chat flow for a better UX.
- **Auto-Scrolling:** The chat window automatically scrolls to the newest message seamlessly.
- **Clear Command:** Users can type `cls` to instantly clear the chat history and start fresh.
- **Extensive Knowledge Base:** Contains answers for MERN Stack, AI/ML, DevOps, Networking, Git, and Databases.

## 🛠️ Tech Stack & Tools

- **Frontend Framework:** React 18
- **Build Tool:** Vite
- **Styling:** Vanilla CSS (Flexbox, custom variables, smooth transitions)
- **State Management:** React Hooks (`useState`, `useEffect`, `useRef`)
- **Key Generation:** Web Crypto API (`crypto.randomUUID()`)

## 🚀 How to Setup the Project

Follow these steps to run the project locally on your machine.

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone <your-repo-link>
   cd codefaq-chatbot-F
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to `http://localhost:5173/` (or the port provided in your terminal) to see the app in action!
